## 2. ТЕОРЕТИЧЕСКАЯ ЧАСТЬ

### 2.1. Операторы циклов в C#

#### Цикл for (цикл с параметром)

Используется, когда заранее известно количество итераций:

```csharp
for (инициализация; условие; изменение)
{
    // тело цикла
}
```

Пример:
```csharp
for (int i = 0; i < n; i++)
{
    sum += i;
}
```

#### Цикл while (цикл с предусловием)

Проверка условия выполняется перед каждой итерацией:

```csharp
while (условие)
{
    // тело цикла
}
```

Пример:
```csharp
while (Math.Abs(term) >= eps)
{
    sum += term;
    term *= x / i;
    i++;
}
```

#### Цикл do-while (цикл с постусловием)

Тело цикла выполняется минимум один раз:

```csharp
do
{
    // тело цикла
}
while (условие);
```

Пример:
```csharp
do
{
    term = Math.Cos(n * x) / (n * n);
    sum += term;
    n++;
}
while (Math.Abs(term) >= eps);
```

### 2.2. Типовые алгоритмы циклической структуры

#### Вычисление суммы n слагаемых

```csharp
double sum = 0;
for (int i = 1; i <= n; i++)
{
    sum += f(i);
}
```

#### Вычисление произведения n сомножителей

```csharp
int product = 1;
for (int i = 1; i <= n; i++)
{
    product *= i;
}
```

#### Вычисление факториала

```csharp
int factorial = 1;
for (int i = 2; i <= n; i++)
{
    factorial *= i;
}
```

#### Табулирование функции

```csharp
for (double x = a; x <= b; x += h)
{
    double y = f(x);
    Console.WriteLine($"{x:f2}\t{y:f4}");
}
```

#### Рекуррентные соотношения

Для ускорения вычислений используют связь между соседними членами ряда:

```csharp
double term = 1.0;
for (int i = 1; i <= n; i++)
{
    term *= x / i;  // переход к следующему члену
    sum += term;
}
```

**Математическое обоснование:**
Ряд Тейлора для косинуса:
cos(x) = 1 - x²/2! + x⁴/4! - x⁶/6! + ...

**Листинг программы:**
```csharp
const double a = 0.1, b = 1.0, h = 0.1;
const double eps = 0.0001;

Console.WriteLine("   x     │    s (ряд)   │   y = cos(x) │  |s - y|  │  n");
Console.WriteLine("───────────────────────────────────────────────────────────");

for (double x = a; x <= b + 0.0001; x += h)
{
    double s = 1.0;
    double term = 1.0;
    int i = 1;

}
```

## 4. ОТВЕТЫ НА КОНТРОЛЬНЫЕ ВОПРОСЫ

### Вопрос 1. Что такое цикл? Операторы цикла for, while, do-while. Различия между ними.

**Ответ:**
Цикл - это управляющая структура, которая позволяет многократно выполнять блок операторов.

В C# существует три основных оператора цикла:

1. **for** - цикл с параметром:
   - Используется при известном числе повторений
   - Управляющая переменная явно указывается в заголовке
   - Синтаксис: for (инициализация; условие; изменение) { }

2. **while** - цикл с предусловием:
   - Проверка условия выполняется перед каждой итерацией
   - Тело может не выполниться ни разу
   - Синтаксис: while (условие) { }

3. **do-while** - цикл с постусловием:
   - Проверка условия после выполнения тела
   - Тело выполняется минимум один раз
   - Синтаксис: do { } while (условие);

**Основные различия:**
- Место проверки условия (до или после выполнения тела)
- Гарантия выполнения (do-while всегда выполняется хотя бы раз)
- Явное управление счетчиком в цикле for

---

### Вопрос 2. Какие данные необходимы для организации цикла for?

**Ответ:**
Для организации цикла for необходимы:

1. **Начальное значение** управляющей переменной (инициализация)
2. **Условие продолжения** цикла (логическое выражение)
3. **Шаг изменения** управляющей переменной (инкремент/декремент)

Пример:
```csharp
for (int i = 0; i < 10; i++)
//    ───┬───  ──┬──  ─┬─
//       │       │     └─ шаг (изменение)
//       │       └─────── условие
//       └─────────────── инициализация
```

Управляющая переменная цикла (счетчик) используется для:
- Отслеживания номера текущей итерации
- Вычисления индексов массивов
- Управления процессом вычислений

### Вопрос 3. Циклы по условию и их организация

**Ответ:**
Циклы по условию (while и do-while) используются, когда:
- Количество итераций заранее неизвестно
- Условие выхода зависит от результатов вычислений
- Требуется достичь определенной точности

**Примеры применения:**

1. **Вычисление рядов с заданной точностью:**
```csharp
do
{
    term = Math.Cos(n * x) / (n * n);
    sum += term;
    n++;
}
while (Math.Abs(term) >= eps);
```

2. **Итерационные методы:**
```csharp
while (Math.Abs(x_new - x_old) > eps)
{
    x_old = x_new;
    x_new = f(x_old);
}
```

3. **Обработка данных до выполнения условия:**
```csharp
while (sum < targetValue)
{
    sum += nextElement();
}
```

### Вопрос 4. Операторы break и continue. В каких случаях они используются?

**Ответ:**

**break** - оператор немедленного выхода из цикла:
```csharp
for (int i = 0; i < 100; i++)
{
    if (found)
        break;  // выход из цикла
}
```

Используется когда:
- Найден искомый элемент в массиве
- Достигнуто специальное условие
- Обнаружена ошибка, требующая прекращения

**continue** - переход к следующей итерации:
```csharp
for (int i = 0; i < n; i++)
{
    if (i % 2 == 0)
        continue;  // пропуск четных чисел
    Process(i);
}
```

Используется когда:
- Нужно пропустить обработку некоторых элементов
- Следует избежать выполнения части кода при особых условиях

### Вопрос 6. Вычисление суммы с использованием рекуррентных соотношений

**Ответ:**
Рекуррентные соотношения позволяют вычислять следующий член ряда через предыдущий, что значительно эффективнее прямого вычисления.

**Пример:** Ряд Тейлора для e^x = Σ(x^i / i!)

**Прямое вычисление (неэффективно):**
```csharp
for (int i = 0; i <= n; i++)
{
    term = Math.Pow(x, i) / Factorial(i);
    sum += term;
}
```

**С рекуррентным соотношением (эффективно):**
```csharp
double term = 1.0;
double sum = 1.0;
for (int i = 1; i <= n; i++)
{
    term *= x / i;  // a[i] = a[i-1] · x / i
    sum += term;
}
```

**Преимущества:**
- Меньше операций (2 вместо возведения в степень и вычисления факториала)
- Выше точность (меньше промежуточных округлений)
- Быстрее выполнение (линейная сложность)

**Вывод рекуррентной формулы:**
```
a[i] = x^i / i!
a[i-1] = x^(i-1) / (i-1)!

a[i] / a[i-1] = x^i / i! · (i-1)! / x^(i-1) = x / i

Следовательно: a[i] = a[i-1] · x / i
```

### Вопрос 7. Вложенные циклы

**Ответ:**
Вложенные циклы - это циклы, расположенные внутри других циклов.

**Структура:**
```csharp
for (внешний цикл)
{
    for (внутренний цикл)
    {
        // тело внутреннего цикла
    }
}
```

**Применение:**

1. **Обработка двумерных массивов:**
```csharp
for (int i = 0; i < rows; i++)
{
    for (int j = 0; j < cols; j++)
    {
        matrix[i][j] = 0;
    }
}
```

2. **Табулирование функций с вычислением рядов:**
```csharp
for (double x = a; x <= b; x += h)  // внешний цикл по x
{
    double sum = 0;
    for (int i = 0; i < n; i++)  // внутренний цикл - ряд
    {
        sum += term(x, i);
    }
    Console.WriteLine($"{x}\t{sum}");
}
```

3. **Генерация комбинаций:**
```csharp
for (int i = 0; i < n; i++)
{
    for (int j = i + 1; j < n; j++)
    {
        // обработка пары (i, j)
    }
}
```

**Важные особенности:**
- Внутренний цикл выполняется полностью для каждой итерации внешнего
- Временная сложность: O(n·m) для двух вложенных циклов
- Необходима правильная инициализация переменных
- Важно следить за областью видимости переменных
