# Лабораторная работа №7: Крестики-нолики

## Описание

Игра "Крестики-нолики" на поле 10x10 с условием победы: 5 символов в ряд.

**Уровень**: 2 (+4 балла)
**Платформа**: Avalonia UI (кроссплатформенная замена Windows Forms)

## Особенности реализации

### Уровень 2 (максимальный балл):
- Игровое поле 10×10 клеток
- Выигрышная комбинация: 5 символов в ряд
- Игра человек vs человек
- Ведение счета игроков
- Визуализация выигрышной комбинации (зеленая линия)
- MessageBox с результатом раунда

### Функционал:
- ✓ Хранение состояния в двумерном массиве `int[10,10]`
- ✓ Определение победы (игрок1, игрок2) или ничьей
- ✓ MessageBox по окончанию раунда
- ✓ Выигрышная комбинация отображается линией
- ✓ Кнопки "Новая игра" и "Сбросить счет"
- ✓ Отрисовка через Canvas
- ✓ Координаты клика определяются через `PointerPressed`

## Класс GameLogic

Отвечает за логику игры:

### Поля:
- `board[10,10]` - игровое поле (0=пусто, 1=X, 2=O)
- `currentPlayer` - текущий игрок (1 или 2)
- `player1Score`, `player2Score` - счет игроков
- `gameOver` - флаг окончания игры

### Методы:
- `MakeMove(row, col)` - сделать ход
- `CheckWin()` - проверка победы (5 в ряд: горизонталь, вертикаль, 2 диагонали)
- `CheckDraw()` - проверка ничьей (все клетки заполнены)
- `ResetGame()` - новая игра
- `ResetScore()` - сброс счета

## Графический интерфейс

### Структура окна:
1. **Верхняя панель** - счет игроков и текущий ход
2. **Canvas 600x600px** - игровое поле с сеткой
3. **Нижние кнопки** - "Новая игра" и "Сбросить счет"

### Отрисовка:
- **Сетка** - черные линии 10x10
- **Крестики (X)** - синие диагональные линии
- **Нолики (O)** - красные круги
- **Выигрышная линия** - зеленая жирная линия через 5 символов

### Обработка событий:
- `OnCanvasClick` - определение клетки по координатам клика
- `OnNewGame` - сброс игры
- `OnResetScore` - сброс счета и игры

## Адаптация Windows Forms → Avalonia

| Windows Forms | Avalonia | Использование |
|---------------|----------|---------------|
| `Paint` event | `Canvas.Children` | Добавление Line, Ellipse |
| `MouseClick` | `PointerPressed` | Обработка кликов |
| `MessageBox.Show()` | Custom Window | Диалоговое окно с результатом |
| `Graphics.DrawLine()` | `new Line()` | Отрисовка линий |
| `Graphics.DrawEllipse()` | `new Ellipse()` | Отрисовка кругов |

## Алгоритм проверки победы

Для каждой клетки проверяются 4 направления:
1. **Горизонталь (→)**: 5 подряд в строке
2. **Вертикаль (↓)**: 5 подряд в столбце
3. **Диагональ (\)**: 5 подряд по диагонали
4. **Диагональ (/)**: 5 подряд по обратной диагонали

При нахождении 5 символов подряд возвращается победитель и список координат выигрышных клеток.

## Особенности реализации на macOS

Windows Forms не поддерживается на macOS, поэтому использован Avalonia UI:
- Полная кроссплатформенность (Windows, macOS, Linux)
- XAML разметка (как в WPF)
- Аналогичный подход к обработке событий
- Canvas для графики

## Управление

1. Кликните на пустую клетку для хода
2. Игроки ходят по очереди (X и O)
3. При победе появляется MessageBox и зеленая линия
4. "Новая игра" - начать новый раунд
5. "Сбросить счет" - обнулить счет и начать заново
