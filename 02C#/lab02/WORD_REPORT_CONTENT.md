## 2. Теоретические сведения

### 2.1. Условный оператор if-else

Условный оператор if позволяет выполнять различные действия в зависимости от выполнения условия.

**Синтаксис:**

```csharp
if (условие)
{
    // Выполняется, если условие истинно
}
else
{
    // Выполняется, если условие ложно
}
```

**Типы разветвлений:**

1. **Полное разветвление** — содержит обе ветви (if-else)
2. **Неполное разветвление (обход)** — содержит только одну ветвь (if без else)
3. **Множественное разветвление** — выбор из нескольких вариантов

### 2.2. Оператор множественного выбора switch

Оператор switch используется для выбора одного из множества вариантов на основе значения переменной.

**Синтаксис:**

```csharp
switch (переменная)
{
    case значение1:
        // Код для случая 1
        break;
    case значение2:
        // Код для случая 2
        break;
    default:
        // Код по умолчанию
        break;
}
```

### 2.3. Логические операторы и выражения

**Основные логические операторы:**

| Оператор | Название | Описание |
|----------|----------|----------|
| && | Логическое И (AND) | Истина, если оба условия истинны |
| \|\| | Логическое ИЛИ (OR) | Истина, если хотя бы одно условие истинно |
| ! | Логическое НЕ (NOT) | Инвертирует значение |

**Операторы сравнения:**

- `==` — равно
- `!=` — не равно
- `>` — больше
- `<` — меньше
- `>=` — больше или равно
- `<=` — меньше или равно

### 2.4. Обработка потока данных

**Случай 1: Количество данных известно**

Используется цикл for с фиксированным числом итераций.

**Случай 2: Количество данных неизвестно**

Используется цикл с условием и специальное значение для завершения ввода (признак конца ввода).

### 3.1. Уровень I: Базовые разветвления

Выполнены 5 задач на освоение условного оператора if-else:

#### Задача 1: Точка на окружности

**Условие:** Определить, лежит ли точка на окружности радиусом r = 2 с центром в начале координат.

**Алгоритм:**
1. Вычислить расстояние от точки до центра: d = √(x² + y²)
2. Проверить условие: |d² - r²| ≤ 10⁻³

**Код решения:**

```csharp
const double r = 2.0;
const double eps = 1e-3;

double distanceSquared = x * x + y * y;
double rSquared = r * r;
double difference = Math.Abs(distanceSquared - rSquared);
bool onCircle = difference <= eps;
```

**Результаты тестирования:**

| Точка (x, y) | Расстояние | Результат |
|--------------|------------|-----------|
| (0.0, 2.0) | 2.0000 | На окружности |
| (1.5, 0.7) | 1.6583 | Вне окружности |
| (1.0, 1.0) | 1.4142 | Вне окружности |
| (3.0, 0.0) | 3.0000 | Вне окружности |

#### Задача 2: Точка в треугольнике

**Условие:** Определить, лежит ли точка внутри треугольника с вершинами (-1, 0), (1, 0), (0, 1).

**Математическое обоснование:**

Точка внутри треугольника, если:
- y ≥ 0 (выше основания)
- y ≤ 1 + x (при x < 0)
- y ≤ 1 - x (при x ≥ 0)

**Код решения:**

```csharp
bool IsInsideTriangle(double x, double y)
{
    if (y < 0) return false;
    return (x < 0) ? (y <= 1 + x) : (y <= 1 - x);
}
```

#### Задача 3: Условный max/min

**Условие:** c = max(a, b), если a ≥ 0 или c = min(a, b), если a < 0

**Код решения:**

```csharp
double c = (a >= 0) ? Math.Max(a, b) : Math.Min(a, b);
```

#### Задача 4: Вложенные операции

**Условие:** z = max(min(a, b), c)

**Алгоритм:**

### 3.2. Уровень II: Разветвления в циклах

Выполнены 3 задачи с фиксированным количеством данных:

#### Задача 10: Студенты без "2" и "3"

**Условие:** Подсчитать студентов, не имеющих оценок "2" и "3" (n студентов, каждый получил 4 оценки).

**Алгоритм:**

```csharp
int studentsWithoutLowGrades = 0;

for (int i = 1; i <= n; i++)
{
    int[] grades = ReadGrades();
    
    bool hasLowGrades = grades.Any(g => g == 2 || g == 3);
    
    if (!hasLowGrades)
        studentsWithoutLowGrades++;
}
```

**Пример результата:**

```
Всего студентов: 3
Студентов без '2' и '3': 2
Процент: 66.7%
```

#### Задача 11: Неуспевающие и средний балл

**Условие:** Подсчитать неуспевающих студентов (имеющих "2") и средний балл группы.

**Формулы:**
- Неуспевающий: хотя бы одна оценка = 2
- Средний балл: Σ(оценки) / (n × 4)

**Результаты:**

```
Всего студентов: 3
Неуспевающих студентов: 1
Средний балл группы: 4.17
```

#### Задача 12: Вычисление площадей

**Условие:** Вычислить площадь выбранной фигуры для n значений.

**Формулы:**
1. Квадрат: S = r²
2. Круг: S = πr²
3. Равносторонний треугольник: S = (r²√3)/4

**Код решения с использованием switch:**

```csharp
double CalculateArea(int figureType, double r)
{
    return figureType switch
    {
        1 => r * r,
        2 => Math.PI * r * r,
        3 => (r * r * Math.Sqrt(3)) / 4,
        _ => throw new ArgumentException()
    };
}
```

### 3.3. Уровень III: Динамический ввод данных

**Особенности реализации:**

## 5. Контрольные вопросы

### Вопрос 1: Что такое разветвление?

**Ответ:** Разветвление — это структура алгоритма, в которой выполнение программы идет по одному из нескольких возможных путей в зависимости от выполнения некоторого условия. Для организации разветвлений используется условный оператор if-else.

### Вопрос 2: Обход и его реализация

**Ответ:** Обход — это частный случай разветвления, когда одна из ветвей не содержит действий. Реализуется с помощью оператора if без else.

### Вопрос 3: Множественный выбор

**Ответ:** Множественный выбор позволяет выбрать одно из множества действий на основе значения переменной. Реализуется с помощью оператора switch или каскада if-else if.

### Вопрос 4: Логические переменные и операторы

**Ответ:** Логические переменные типа bool принимают значения true или false. Логические операторы (&&, ||, !) используются для построения сложных условий.

### Вопрос 5: Обработка потока данных

**Ответ:** При неизвестном количестве данных используются циклы с условием и специальное значение (признак конца ввода) для завершения обработки.

### Вопрос 6: Ввод нескольких значений

**Ответ:** При вводе нескольких связанных значений для организации цикла выбирается одно значение в качестве контрольного для проверки признака конца ввода.

### Вопрос 7: Реализация множественного выбора

**Ответ:** Множественный выбор реализуется оператором switch (для простых случаев) или каскадом if-else if (для сложных условий).
