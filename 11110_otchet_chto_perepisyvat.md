# Часть 1. §5.3 — двойные интегралы (пластина в плоскости Oxy)

## §5.3 — Задача 1

**Условие (переписать дословно из своего листа варианта):** границы \(y = Ax^3\), \(y = Bx\), \(x \ge 0\); у тебя \(A=1\), \(B=4\). Плотность \(\Gamma(x,y) = Cy = 6y\).

**────────────────────────────────────────────────────**  
**[ВСТАВИТЬ РИСУНОК ИЗ GEOGEBRA — ЗДЕСЬ]**  
**Что должно быть на рисунке:** плоскость Oxy, кривые \(y=x^3\) и \(y=4x\), заштрихованная область **D** между ними (для \(0 \le x \le 2\)). Желательно отметить точку пересечения \((2,\,8)\).  
**GeoGebra (2D):** `y = x^3`, `y = 4 x`.  
**Подпись к рисунку:** например, *Рис. 1. Область D (§5.3, задача 1).*  
**────────────────────────────────────────────────────**

**Решение:** запиши пределы интегрирования и интегралы для **S**, **m**, **M_y**, **M_x**, затем **x̄ = M_y/m**, **ȳ = M_x/m**. Итоговая таблица чисел — как в `solution_11110.md` (§5.3-1).

---

## §5.3 — Задача 2

**Условие:** \(x^2+y^2 = Ax\) при \(A=4\); прямые \(y=x\), \(y=4x\); ограничение \(x \le y \le 4x\). Плотность \(\Gamma = 6\) (постоянная).

**────────────────────────────────────────────────────**  
**[ВСТАВИТЬ РИСУНОК ИЗ GEOGEBRA — ЗДЕСЬ]**  
**Что должно быть на рисунке:** окружность \((x-2)^2+y^2=4\), прямые \(y=x\) и \(y=4x\), заштрихованный **клин** внутри круга между этими прямыми (первая четверть, «сектор» между лучами).  
**GeoGebra (2D):** `x^2 + y^2 = 4 x`, `y = x`, `y = 4 x`.  
**Подпись к рисунку:** например, *Рис. 2. Область D (§5.3, задача 2).*  
**────────────────────────────────────────────────────**

**Решение:** переход к полярным координатам, **S**, **m**, **M_y**, **M_x**, **x̄**, **ȳ** — по `solution_11110.md` (§5.3-2).

# Часть 2. §5.4 — тройные интегралы (тело в пространстве)

## §5.4 — Задача 1 (конус и цилиндр, y ≤ 0)

**Условие:** \(z^2 = A(x^2+y^2)\), \(x^2+y^2 = B^2\), \(z \ge 0\), \(y \le 0\); \(\Gamma = C\sqrt{x^2+y^2}\); \(A=1\), \(B=3\), \(C=6\).

**────────────────────────────────────────────────────**  
**[ВСТАВИТЬ РИСУНОК ИЗ GEOGEBRA — ЗДЕСЬ]** (основной, 3D)  
**Что должно быть на рисунке:** верхняя полость конуса \(z=\sqrt{x^2+y^2}\) и цилиндр радиуса 3; область с **y ≤ 0** (удобно ориентировать камеру «снизу» или смотреть на полупространство за плоскостью \(y=0\)).  
**GeoGebra (3D):** `z = sqrt(x^2 + y^2)`, `Cylinder((0, 0, 0), (0, 0, 1), 3)`, `y = 0`.  

**[ПО ЖЕЛАНИЮ / ЕСЛИ ПРОСЯТ — ВТОРОЙ РИСУНОК ЗДЕСЬ]** — вид **сверху** (2D): `x^2 + y^2 = 9`, `y = 0`, заштриховать полукруг или сектор с **φ от π до 2π** (нижние квадранты).  
**Подпись:** *Рис. 3. Область G (§5.4, задача 1).*  
**────────────────────────────────────────────────────**

**Решение:** цилиндрические координаты, **V, m**, моменты, центр масс, **I_z** — см. `solution_11110.md`.

## §5.4 — Задача 2 (часть шара: z ≥ 0, x ≥ 0)

**Условие:** \(x^2+y^2+z^2 = A^2\), \(z \ge 0\), \(x \ge 0\); \(\Gamma = C\sqrt{x^2+y^2+z^2} = C\rho\); \(A=5\), \(C=6\).

**────────────────────────────────────────────────────**  
**[ВСТАВИТЬ РИСУНОК ИЗ GEOGEBRA — ЗДЕСЬ]**  
**Что должно быть на рисунке:** шар радиуса 5, видна **четверть** «верхней» половины со стороны **x > 0** (можно показать сферу + плоскости \(z=0\) и \(x=0\) как границы).  
**GeoGebra (3D):** `Sphere((0, 0, 0), 5)`, `z = sqrt(25 - x^2 - y^2)`, `z = 0`, `x = 0`.  
**Подпись:** *Рис. 4. Область G (§5.4, задача 2).*  
**────────────────────────────────────────────────────**

**Решение:** сферические координаты, **V, m**, моменты, центр масс, **I_z** — см. `solution_11110.md`.
