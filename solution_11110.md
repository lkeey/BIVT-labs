# Решение задачи 11110: Приложения тройных интегралов

## Задача 1: Область конус-цилиндр

### Условие:
- Область G: Z² = A(X² + Y²), X² + Y² = B², Z ≥ 0, Y ≤ 0
- Плотность: Γ(x,y,z) = C√(X² + Y²)
- Параметры: A = 1.0, B = 3.0, C = 6.0

### Геометрия области:
- Конус: Z² = X² + Y² → Z = √(X² + Y²) (т.к. Z ≥ 0)
- Цилиндр: X² + Y² = 9 (радиус 3)
- Условие Y ≤ 0 означает нижнюю полуплоскость (III и IV квадранты)

### Переход к цилиндрическим координатам:
- x = r cos φ
- y = r sin φ
- z = z
- dV = r dr dφ dz
- Плотность: Γ = 6r

### Пределы интегрирования:
- r: от 0 до 3
- φ: от π до 2π (условие Y ≤ 0)
- z: от 0 до r (от основания до поверхности конуса)

---

## Вычисления для Задачи 1:

### 1. Объем V

V = ∫∫∫ dV = ∫[π→2π] ∫[0→3] ∫[0→r] r dz dr dφ

**Шаг 1:** Интегрирование по z:
∫[0→r] r dz = r·z |[0→r] = r·r = r²

**Шаг 2:** Интегрирование по r:
∫[0→3] r² dr = r³/3 |[0→3] = 27/3 = 9

**Шаг 3:** Интегрирование по φ:
∫[π→2π] 9 dφ = 9φ |[π→2π] = 9(2π - π) = 9π

**Ответ:** V = 9π ≈ 28.274

---

### 2. Масса m

m = ∫∫∫ Γ dV = ∫[π→2π] ∫[0→3] ∫[0→r] 6r·r dz dr dφ = ∫[π→2π] ∫[0→3] ∫[0→r] 6r² dz dr dφ

**Шаг 1:** Интегрирование по z:
∫[0→r] 6r² dz = 6r²·z |[0→r] = 6r³

**Шаг 2:** Интегрирование по r:
∫[0→3] 6r³ dr = 6·r⁴/4 |[0→3] = (3/2)·81 = 121.5

**Шаг 3:** Интегрирование по φ:
∫[π→2π] 121.5 dφ = 121.5φ |[π→2π] = 121.5π

**Ответ:** m = 121.5π ≈ 381.704

---

### 3. Статический момент M_yz (относительно плоскости Oyz)

M_yz = ∫∫∫ x·Γ dV = ∫[π→2π] ∫[0→3] ∫[0→r] (r cos φ)·6r·r dz dr dφ

= ∫[π→2π] ∫[0→3] ∫[0→r] 6r³ cos φ dz dr dφ

**Шаг 1:** Интегрирование по z:
∫[0→r] 6r³ cos φ dz = 6r³ cos φ·z |[0→r] = 6r⁴ cos φ

**Шаг 2:** Интегрирование по r:
∫[0→3] 6r⁴ cos φ dr = 6 cos φ·r⁵/5 |[0→3] = (6/5)·243·cos φ = 291.6 cos φ

**Шаг 3:** Интегрирование по φ:
∫[π→2π] 291.6 cos φ dφ = 291.6 sin φ |[π→2π] = 291.6(sin 2π - sin π) = 291.6(0 - 0) = 0

**Ответ:** M_yz = 0

---

### 4. Статический момент M_xz (относительно плоскости Oxz)

M_xz = ∫∫∫ y·Γ dV = ∫[π→2π] ∫[0→3] ∫[0→r] (r sin φ)·6r·r dz dr dφ

= ∫[π→2π] ∫[0→3] ∫[0→r] 6r³ sin φ dz dr dφ

**Шаг 1:** Интегрирование по z:
∫[0→r] 6r³ sin φ dz = 6r⁴ sin φ

**Шаг 2:** Интегрирование по r:
∫[0→3] 6r⁴ sin φ dr = 291.6 sin φ

**Шаг 3:** Интегрирование по φ:
∫[π→2π] 291.6 sin φ dφ = -291.6 cos φ |[π→2π] = -291.6(cos 2π - cos π) = -291.6(1 - (-1)) = -291.6·2 = -583.2

**Ответ:** M_xz = -583.2

---

### 5. Статический момент M_xy (относительно плоскости Oxy)

M_xy = ∫∫∫ z·Γ dV = ∫[π→2π] ∫[0→3] ∫[0→r] z·6r·r dz dr dφ

= ∫[π→2π] ∫[0→3] ∫[0→r] 6r²z dz dr dφ

**Шаг 1:** Интегрирование по z:
∫[0→r] 6r²z dz = 6r²·z²/2 |[0→r] = 3r⁴

**Шаг 2:** Интегрирование по r:
∫[0→3] 3r⁴ dr = 3·r⁵/5 |[0→3] = (3/5)·243 = 145.8

**Шаг 3:** Интегрирование по φ:
∫[π→2π] 145.8 dφ = 145.8π

**Ответ:** M_xy = 145.8π ≈ 458.149

---

### 6. Координаты центра масс

x_c = M_yz / m = 0 / (121.5π) = 0

y_c = M_xz / m = -583.2 / (121.5π) = -583.2 / 381.704 ≈ -1.528

z_c = M_xy / m = 145.8π / (121.5π) = 145.8 / 121.5 = 1.2

**Ответ:** 
- x_c = 0
- y_c ≈ -1.528
- z_c = 1.2

---

### 7. Момент инерции I_z (относительно оси Oz)

I_z = ∫∫∫ (x² + y²)·Γ dV = ∫[π→2π] ∫[0→3] ∫[0→r] r²·6r·r dz dr dφ

= ∫[π→2π] ∫[0→3] ∫[0→r] 6r⁴ dz dr dφ

**Шаг 1:** Интегрирование по z:
∫[0→r] 6r⁴ dz = 6r⁴·z |[0→r] = 6r⁵

**Шаг 2:** Интегрирование по r:
∫[0→3] 6r⁵ dr = 6·r⁶/6 |[0→3] = 729

**Шаг 3:** Интегрирование по φ:
∫[π→2π] 729 dφ = 729π

**Ответ:** I_z = 729π ≈ 2290.221

---

## Итоговые результаты для Задачи 1:

1. **V (объем)** = 9π ≈ **28.274**
2. **m (масса)** = 121.5π ≈ **381.704**
3. **M_yz** = **0**
4. **M_xz** ≈ **-583.2**
5. **M_xy** = 145.8π ≈ **458.149**
6. **Координаты центра масс:**
   - x_c = **0**
   - y_c ≈ **-1.528**
   - z_c = **1.2**
7. **I_z (момент инерции)** = 729π ≈ **2290.221**

## Задача 2: Область полусфера

### Условие:
- Область G: X² + Y² + Z² = A², Z ≥ 0, X ≥ 0
- Плотность: Γ(x,y,z) = C√(X² + Y² + Z²)
- Параметры: A = 5.0, C = 6.0

### Геометрия области:
- Сфера радиуса 5: X² + Y² + Z² = 25
- Верхняя полусфера: Z ≥ 0
- Правая половина: X ≥ 0

### Переход к сферическим координатам:
- x = ρ sin θ cos φ
- y = ρ sin θ sin φ
- z = ρ cos θ
- dV = ρ² sin θ dρ dφ dθ
- Плотность: Γ = 6ρ

### Пределы интегрирования:
- ρ: от 0 до 5
- θ: от 0 до π/2 (условие Z ≥ 0)
- φ: от -π/2 до π/2 (условие X ≥ 0, т.е. cos φ ≥ 0)

---

## Вычисления для Задачи 2:

### 1. Объем V

V = ∫∫∫ dV = ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] ρ² sin θ dρ dθ dφ

**Шаг 1:** Интегрирование по ρ:
∫[0→5] ρ² dρ = ρ³/3 |[0→5] = 125/3

**Шаг 2:** Интегрирование по θ:
∫[0→π/2] (125/3) sin θ dθ = (125/3)(-cos θ) |[0→π/2] = (125/3)(0 - (-1)) = 125/3

**Шаг 3:** Интегрирование по φ:
∫[-π/2→π/2] 125/3 dφ = (125/3)φ |[-π/2→π/2] = (125/3)·π

**Ответ:** V = 125π/3 ≈ 130.900

---

### 2. Масса m

m = ∫∫∫ Γ dV = ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] 6ρ·ρ² sin θ dρ dθ dφ

= ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] 6ρ³ sin θ dρ dθ dφ

**Шаг 1:** Интегрирование по ρ:
∫[0→5] 6ρ³ dρ = 6·ρ⁴/4 |[0→5] = (3/2)·625 = 937.5

**Шаг 2:** Интегрирование по θ:
∫[0→π/2] 937.5 sin θ dθ = -937.5 cos θ |[0→π/2] = 937.5

**Шаг 3:** Интегрирование по φ:
∫[-π/2→π/2] 937.5 dφ = 937.5π

**Ответ:** m = 937.5π ≈ 2945.243

---

### 3. Статический момент M_yz (относительно плоскости Oyz)

M_yz = ∫∫∫ x·Γ dV = ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] (ρ sin θ cos φ)·6ρ·ρ² sin θ dρ dθ dφ

= ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] 6ρ⁴ sin² θ cos φ dρ dθ dφ

**Шаг 1:** Интегрирование по ρ:
∫[0→5] 6ρ⁴ sin² θ cos φ dρ = 6 sin² θ cos φ·ρ⁵/5 |[0→5] = (6/5)·3125·sin² θ cos φ = 3750 sin² θ cos φ

**Шаг 2:** Интегрирование по θ (используем ∫ sin² θ dθ = θ/2 - sin(2θ)/4):
∫[0→π/2] 3750 sin² θ cos φ dθ = 3750 cos φ·[θ/2 - sin(2θ)/4] |[0→π/2]
= 3750 cos φ·[(π/4 - 0) - (0 - 0)] = 3750 cos φ·π/4 = 937.5π cos φ

**Шаг 3:** Интегрирование по φ:
∫[-π/2→π/2] 937.5π cos φ dφ = 937.5π sin φ |[-π/2→π/2] = 937.5π(1 - (-1)) = 1875π

**Ответ:** M_yz = 1875π ≈ 5890.486

---

### 4. Статический момент M_xz (относительно плоскости Oxz)

M_xz = ∫∫∫ y·Γ dV = ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] (ρ sin θ sin φ)·6ρ·ρ² sin θ dρ dθ dφ

= ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] 6ρ⁴ sin² θ sin φ dρ dθ dφ

**Шаг 1:** Интегрирование по ρ:
∫[0→5] 6ρ⁴ dρ = 3750

**Шаг 2:** Интегрирование по θ:
∫[0→π/2] 3750 sin² θ sin φ dθ = 3750 sin φ·π/4 = 937.5π sin φ

**Шаг 3:** Интегрирование по φ:
∫[-π/2→π/2] 937.5π sin φ dφ = -937.5π cos φ |[-π/2→π/2] = -937.5π(0 - 0) = 0

**Ответ:** M_xz = 0

---

### 5. Статический момент M_xy (относительно плоскости Oxy)

M_xy = ∫∫∫ z·Γ dV = ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] (ρ cos θ)·6ρ·ρ² sin θ dρ dθ dφ

= ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] 6ρ⁴ cos θ sin θ dρ dθ dφ

**Шаг 1:** Интегрирование по ρ:
∫[0→5] 6ρ⁴ cos θ sin θ dρ = 3750 cos θ sin θ

**Шаг 2:** Интегрирование по θ (используем ∫ cos θ sin θ dθ = sin² θ / 2):
∫[0→π/2] 3750 cos θ sin θ dθ = 3750·sin² θ/2 |[0→π/2] = 3750·(1/2) = 1875

**Шаг 3:** Интегрирование по φ:
∫[-π/2→π/2] 1875 dφ = 1875π

**Ответ:** M_xy = 1875π ≈ 5890.486

---

### 6. Координаты центра масс

x_c = M_yz / m = 1875π / (937.5π) = 1875 / 937.5 = 2.0

y_c = M_xz / m = 0 / (937.5π) = 0

z_c = M_xy / m = 1875π / (937.5π) = 1875 / 937.5 = 2.0

**Ответ:** 
- x_c = 2.0
- y_c = 0
- z_c = 2.0

---

### 7. Момент инерции I_z (относительно оси Oz)

I_z = ∫∫∫ (x² + y²)·Γ dV = ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] (ρ² sin² θ)·6ρ·ρ² sin θ dρ dθ dφ

= ∫[-π/2→π/2] ∫[0→π/2] ∫[0→5] 6ρ⁵ sin³ θ dρ dθ dφ

**Шаг 1:** Интегрирование по ρ:
∫[0→5] 6ρ⁵ dρ = 6·ρ⁶/6 |[0→5] = 15625

**Шаг 2:** Интегрирование по θ (используем ∫ sin³ θ dθ = -cos θ + cos³ θ/3):
∫[0→π/2] 15625 sin³ θ dθ = 15625·[-cos θ + cos³ θ/3] |[0→π/2]
= 15625·[(0 - 0) - (-1 + (-1)³/3)]
= 15625·[0 - (-1 - 1/3)]
= 15625·[0 + 4/3]
= 15625·4/3
= 20833.333...

**Шаг 3:** Интегрирование по φ:
∫[-π/2→π/2] 20833.333 dφ = 20833.333·π

**Ответ:** I_z = (62500/3)π ≈ 65449.847

---

## Итоговые результаты для Задачи 2:

1. **V (объем)** = 125π/3 ≈ **130.900**
2. **m (масса)** = 937.5π ≈ **2945.243**
3. **M_yz** = 1875π ≈ **5890.486**
4. **M_xz** = **0**
5. **M_xy** = 1875π ≈ **5890.486**
6. **Координаты центра масс:**
