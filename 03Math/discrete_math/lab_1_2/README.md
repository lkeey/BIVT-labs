# Лабораторная работа 1.2: Прямое произведение, графики, уравнения

## Описание

Решение заданий по дискретной математике на тему "Прямое произведение, графики, уравнения".

**Вариант:** 10

## Содержание работы

### Задание 1.2.1
Проверка и доказательство равенства множеств через принадлежность элементов:
```
A×(B∪C) = (A×(B∆C))∪(A×(B∩C))
```

**Результат:** Равенство доказано для произвольных множеств A, B, C методом двойного включения.

### Задание 1.2.2
Операции над графиками для P = {(0,2), (2,3), (3,3), (3,0), (0,0)}:
- Инверсия P⁻¹
- Композиция P ∘ P
- Композиция P⁻¹ ∘ P
- Декартово произведение проекций

### Задание 1.2.3
Решение уравнения относительно графика X:
```
X ∘ P = T
```
при условиях |X| = 6, пр₁X = пр₂X = {1,2,3,4,5,6}

**Результат:** Найдены 2 решения с вычислением композиций P ∘ X для каждого.
