## Теоретические определения

**Декартово произведение** множеств A₁, A₂,..., Aₙ называется множество:
```
A₁ × A₂ × ... × Aₙ = {(a₁, a₂,..., aₙ) | a₁ ∈ A₁,..., aₙ ∈ Aₙ}
```

**Графиком** называется подмножество декартова произведения двух множеств.

**Инверсией графика** P называется график:
```
P⁻¹ = {(a,b) | (b,a) ∈ P}
```

**Композицией графиков** P и Q называется график:
```
P ∘ Q = {(a,b) | ∃ₓ ((a,x) ∈ P и (x,b) ∈ Q)}
```

## Задание 1.2.1

### Условие
**Дано:** A = {1,2}, B = {2,3}, C = {1,3}

**Требуется:**
1. Проверить справедливость равенства: **A×(B∪C) = (A×(B∆C))∪(A×(B∩C))**
2. Доказать через принадлежность произвольного элемента
3. Выяснить, верно ли для произвольных множеств A, B, C

---

### Часть 1: Проверка на конкретном примере

**Вычисление левой части A×(B∪C):**

```
B∪C = {2,3} ∪ {1,3} = {1,2,3}

A×(B∪C) = {1,2} × {1,2,3} = {(1,1), (1,2), (1,3), (2,1), (2,2), (2,3)}
```

**Вычисление правой части (A×(B∆C))∪(A×(B∩C)):**

Сначала найдем необходимые множества:

```
B∩C = {2,3} ∩ {1,3} = {3}

B∆C (симметрическая разность) = (B∪C) \ (B∩C) = {1,2,3} \ {3} = {1,2}
```

Теперь вычислим декартовы произведения:

```
A×(B∆C) = {1,2} × {1,2} = {(1,1), (1,2), (2,1), (2,2)}

A×(B∩C) = {1,2} × {3} = {(1,3), (2,3)}

(A×(B∆C))∪(A×(B∩C)) = {(1,1), (1,2), (2,1), (2,2)} ∪ {(1,3), (2,3)}
                      = {(1,1), (1,2), (1,3), (2,1), (2,2), (2,3)}
```

**Сравнение:**
```
Левая часть:  A×(B∪C) = {(1,1), (1,2), (1,3), (2,1), (2,2), (2,3)}
Правая часть: (A×(B∆C))∪(A×(B∩C)) = {(1,1), (1,2), (1,3), (2,1), (2,2), (2,3)}
```

**Вывод:** Для данных множеств равенство **выполняется** ✓

---

### Часть 2: Доказательство через принадлежность элемента

**Теорема:** Для произвольных множеств A, B, C справедливо равенство:
```
A×(B∪C) = (A×(B∆C))∪(A×(B∩C))
```

**Доказательство:**

Докажем, что произвольный элемент (a,b) принадлежит левой части тогда и только тогда, когда он принадлежит правой части.

**Направление (⇒): A×(B∪C) ⊆ (A×(B∆C))∪(A×(B∩C))**

Пусть (a,b) ∈ A×(B∪C).

Тогда по определению декартова произведения:
```
a ∈ A  и  b ∈ (B∪C)
```

Из b ∈ (B∪C) следует, что **b ∈ B или b ∈ C** (или оба условия).

Рассмотрим все возможные случаи:

**Случай 1:** b ∈ B и b ∉ C
- Тогда b ∈ B∆C (по определению симметрической разности)
- Следовательно, (a,b) ∈ A×(B∆C)
- Следовательно, (a,b) ∈ (A×(B∆C))∪(A×(B∩C))

**Случай 2:** b ∉ B и b ∈ C
- Тогда b ∈ B∆C (по определению симметрической разности)
- Следовательно, (a,b) ∈ A×(B∆C)
- Следовательно, (a,b) ∈ (A×(B∆C))∪(A×(B∩C))

**Случай 3:** b ∈ B и b ∈ C
- Тогда b ∈ B∩C (по определению пересечения)
- Следовательно, (a,b) ∈ A×(B∩C)
- Следовательно, (a,b) ∈ (A×(B∆C))∪(A×(B∩C))

Во всех случаях (a,b) ∈ (A×(B∆C))∪(A×(B∩C)). ✓

---

**Направление (⇐): (A×(B∆C))∪(A×(B∩C)) ⊆ A×(B∪C)**

Пусть (a,b) ∈ (A×(B∆C))∪(A×(B∩C)).

Тогда (a,b) ∈ A×(B∆C) или (a,b) ∈ A×(B∩C).

**Случай 1:** (a,b) ∈ A×(B∆C)
- Тогда a ∈ A и b ∈ B∆C
- По определению B∆C: b ∈ (B∪C) \ (B∩C)
- Следовательно, b ∈ B∪C
- Следовательно, (a,b) ∈ A×(B∪C)

**Случай 2:** (a,b) ∈ A×(B∩C)
- Тогда a ∈ A и b ∈ B∩C
- Из b ∈ B∩C следует b ∈ B и b ∈ C
- Следовательно, b ∈ B∪C
- Следовательно, (a,b) ∈ A×(B∪C)

В обоих случаях (a,b) ∈ A×(B∪C). ✓

---

**Заключение:**

Мы доказали оба включения:
- A×(B∪C) ⊆ (A×(B∆C))∪(A×(B∩C))
- (A×(B∆C))∪(A×(B∩C)) ⊆ A×(B∪C)

Следовательно, **A×(B∪C) = (A×(B∆C))∪(A×(B∩C))** для произвольных множеств A, B, C. ∎

**Ответ:** Равенство **верно для произвольных множеств** A, B, C.

## Задание 1.2.2

### Условие
**Дано:** P = {(0,2), (2,3), (3,3), (3,0), (0,0)}

**Требуется найти:**
1. P⁻¹ — инверсия графика
2. P ∘ P — композиция
3. P⁻¹ ∘ P
4. пр₂(P⁻¹ ∘ P) × пр₁(P ∘ P)

---

### Решение

**1. Инверсия P⁻¹:**

По определению инверсии меняем местами компоненты каждой пары:

```
P⁻¹ = {(b,a) | (a,b) ∈ P}
```

Результат:
```
P⁻¹ = {(2,0), (3,2), (3,3), (0,3), (0,0)}
```

---

**2. Композиция P ∘ P:**

По определению композиции:
```
(a,c) ∈ P ∘ P ⟺ ∃x: (a,x) ∈ P и (x,c) ∈ P
```

Систематически проверим все возможные комбинации:

| Первая пара (a,x) ∈ P | Вторая пара (x,c) ∈ P | Результат (a,c) |
|------------------------|------------------------|-----------------|
| (0,2) | (2,3) | (0,3) ✓ |
| (0,0) | (0,2) | (0,2) ✓ |
| (0,0) | (0,0) | (0,0) ✓ |
| (2,3) | (3,3) | (2,3) ✓ |
| (2,3) | (3,0) | (2,0) ✓ |
| (3,3) | (3,3) | (3,3) ✓ |
| (3,3) | (3,0) | (3,0) ✓ |
| (3,0) | (0,2) | (3,2) ✓ |
| (3,0) | (0,0) | (3,0) (дубликат) |

Результат:
```
P ∘ P = {(0,0), (0,2), (0,3), (2,0), (2,3), (3,0), (3,2), (3,3)}
```

---

**3. Композиция P⁻¹ ∘ P:**

Используем P⁻¹ = {(2,0), (3,2), (3,3), (0,3), (0,0)} и P = {(0,2), (2,3), (3,3), (3,0), (0,0)}

По определению композиции ищем пары (a,c) такие, что ∃x: (a,x) ∈ P⁻¹ и (x,c) ∈ P

| Первая пара (a,x) ∈ P⁻¹ | Вторая пара (x,c) ∈ P | Результат (a,c) |
|--------------------------|------------------------|-----------------|
| (2,0) | (0,2) | (2,2) ✓ |
| (2,0) | (0,0) | (2,0) ✓ |
| (3,2) | (2,3) | (3,3) ✓ |
| (3,3) | (3,3) | (3,3) (дубликат) |
| (3,3) | (3,0) | (3,0) ✓ |
| (0,3) | (3,3) | (0,3) ✓ |
| (0,3) | (3,0) | (0,0) ✓ |
| (0,0) | (0,2) | (0,2) ✓ |
| (0,0) | (0,0) | (0,0) (дубликат) |

Результат:
```
P⁻¹ ∘ P = {(0,0), (0,2), (0,3), (2,0), (2,2), (3,0), (3,3)}
```

---

**4. Декартово произведение проекций пр₂(P⁻¹ ∘ P) × пр₁(P ∘ P):**

Сначала найдем проекции:

**Проекция на вторую координату:**
```
пр₂(P⁻¹ ∘ P) = {b | (a,b) ∈ P⁻¹ ∘ P}
             = {0, 2, 3}
```

**Проекция на первую координату:**
```
пр₁(P ∘ P) = {a | (a,b) ∈ P ∘ P}
           = {0, 2, 3}
```

**Декартово произведение:**
```
пр₂(P⁻¹ ∘ P) × пр₁(P ∘ P) = {0, 2, 3} × {0, 2, 3}
                           = {(0,0), (0,2), (0,3),
                              (2,0), (2,2), (2,3),
                              (3,0), (3,2), (3,3)}
```

---

### Ответы к заданию 1.2.2:

1. **P⁻¹** = {(2,0), (3,2), (3,3), (0,3), (0,0)}

2. **P ∘ P** = {(0,0), (0,2), (0,3), (2,0), (2,3), (3,0), (3,2), (3,3)}

3. **P⁻¹ ∘ P** = {(0,0), (0,2), (0,3), (2,0), (2,2), (3,0), (3,3)}

4. **пр₂(P⁻¹ ∘ P) × пр₁(P ∘ P)** = {(0,0), (0,2), (0,3), (2,0), (2,2), (2,3), (3,0), (3,2), (3,3)}

---

### Решение

**Анализ уравнения X ∘ P = T:**

Композиция (a,c) ∈ X ∘ P означает: ∃b: (a,b) ∈ X и (b,c) ∈ P

Для каждой пары (a,c) ∈ T нужно найти такое b, чтобы:
- (a,b) ∈ X
- (b,c) ∈ P

Другими словами, для каждой пары из T ищем промежуточную точку через P.

---

**Шаг 1: Построение множества кандидатов**

Для каждой пары (a,c) ∈ T ищем все пары (b,c) ∈ P (пары с той же второй координатой).
Тогда (a,b) — кандидат для включения в X.

**Для (2,2) ∈ T:** ищем пары (?,2) в P
- (4,2) ∈ P → кандидат (2,4)
- (5,2) ∈ P → кандидат (2,5)

**Для (5,3) ∈ T:** ищем пары (?,3) в P

**Для (4,6) ∈ T:** ищем пары (?,6) в P

```

```

**Шаг 2: Формирование графика X**

Нам нужно выбрать из множества A такое подмножество X, чтобы:
1. X ∘ P = T (гарантировано покрытие всех пар из T)
2. |X| = 6
3. пр₁X = {1,2,3,4,5,6}
4. пр₂X = {1,2,3,4,5,6}
